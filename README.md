# Solar Shanty — play in browser

A mirror of `SolarShanty.html`, the current single-file build of
**Solar Shanty**, a first-person solar houseboat life simulator. Source
lives in a private repo; this one exists only to serve that one file
publicly via GitHub Pages.

**Play:** https://whitneydesignlabs.github.io/solar-shanty-play/

`index.html` is a one-line redirect to `SolarShanty.html` — GitHub Pages
needs a file with that exact name at the root to serve the bare URL above.
`SolarShanty.html` itself is the real artifact and keeps its canonical
name so it can be diffed/synced directly against the private repo's copy.

**Keeping this in sync:** whenever the private repo's `SolarShanty.html`
is refreshed, copy it here verbatim (same filename) and push — no other
changes needed. The embedded version stamp (Esc → Credits, in-game) is
the fastest way to check whether this mirror is behind.

No install, no account — runs entirely in your browser. Progress autosaves
locally each game hour.
