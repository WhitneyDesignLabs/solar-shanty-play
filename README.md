# Solar Shanty — play in browser

A single-file mirror of the current build of **Solar Shanty**, a first-person
solar houseboat life simulator. Source lives in a private repo; this one
exists only to serve the built `index.html` via GitHub Pages.

**Play:** https://whitneydesignlabs.github.io/solar-shanty-play/

No install, no account — runs entirely in your browser. Progress autosaves
locally each game hour.
